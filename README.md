# Beautiful Mermaid Editor

A live editor for [beautiful-mermaid](https://github.com/lukilabs/beautiful-mermaid) â€” render Mermaid diagrams as beautiful SVGs or ASCII art.

## Features

- Live preview with debounced rendering
- SVG output with 15 built-in themes
- ASCII and Unicode text output for terminals
- Transparent background mode
- Copy and download outputs
- Resizable split panes
- Code persistence via localStorage
- Monaco editor with syntax highlighting

## Getting Started

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
